# azure-dio
Como criar um Projeto de Machine Learning na Azure


-  Configurar uma Conta na Azure
Se ainda não tiver uma conta na Azure, crie uma em https://azure.microsoft.com/

 - Já com a conta criada e logada
Clique em "Criar um Recurso" e selecione "Machine Learning"
(Selecione ou coloque na busca Azure Machine Learning e clique nele)
Aparecerá uma opção de Plano que será a Azure Machine Learning. Clique em Create

- Na próxima tela, selecione: 'subscription e resource group"
A Subscription será a default (Azure subscription 1)
No caso do Resource Group, será um nome relacionado ao seu modelo de Machine Learning
Caso não tenha nenhum grupo, clicar em Create New e criar um novo grupo
Após isto slecione o nome do Workspace, a Região e os demais campos serão preenchidas
automaticamente. Caso queira alterar algum campo, é só fazer a alteração antes de revisar

Estando tudo ok, clicar em review e posteriormente em create

- O Deployment ficará em progresso até terminar de criar seu local de trabalho para gerenciamento
dos modelos

Após finalizar clique no botão Go To Resource que lhe permitará trabalhar com os modelos  

- Você será redirecioado para uma tela (o Azure Machine Learning Studio)

Clique em Launch Studio e você será redirecionado para uma nova página 

- Será solicitado que seja criado seu Workspace

Após isto, na barra lateral esquerda clique em ML Automatizado (FIGURA PIC4)

Na tela do ML Automatizado, clique em novo trabalho de ML Automatizado  (FIGURA PIC5)


Siga o passo a passo:

(FIGURA PIC6, PIC7, PIC8, PIC9, PIC10)

Confira os campos e complete o campos  de descrição. Clique em Avançar 


Selecione o Tipo de Tarefa: (FIGURA PIC11)
- Classificação 
- Regressão
- Previsão de Série Temporal
- Processamento de Linguagem Natural ou 
- Pesquisa Visual Computacional

e Clique em Criar. Será solicitado o tipo de dados e a fonte dos mesmo

Confirme as configurações do arquivo de origem dos dados

Confirme o formato dos dados

Finalmente clique em criar
